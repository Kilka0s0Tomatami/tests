Тест-сьют: Авторизация
№	Название теста	Описание	Шаги	Ожидаемый результат
1	Reg New User	Регистрация нового пользователя	1. Перейти на страницу регистрации. 2. Ввести валидные данные (email или телефон). 3. Подтвердить код из письма/SMS. 4. Нажать "Зарегистрироваться".	Пользователь успешно зарегистрирован.
2	Failed Reg User Again	Попытка повторной регистрации с тем же email	1. Перейти на страницу регистрации. 2. Ввести email, который уже зарегистрирован. 3. Ввести пароль. 4. Нажать "Зарегистрироваться".	Ошибка: "Этот email уже зарегистрирован".
3	Login User	Авторизация зарегистрированного пользователя	1. Перейти на страницу авторизации. 2. Ввести корректный email и пароль. 3. Нажать "Войти".	Успешный вход в систему.
4	Failed Login User With Wrong Password	Попытка входа с неверным паролем	1. Перейти на страницу авторизации. 2. Ввести email и неправильный пароль. 3. Нажать "Войти".	Ошибка: "Неверный логин или пароль".
5	Failed Login With Unexistent User	Попытка входа с несуществующим пользователем	1. Перейти на страницу авторизации. 2. Ввести email, который не зарегистрирован. 3. Нажать "Войти".	Ошибка: "Аккаунт не найден".
